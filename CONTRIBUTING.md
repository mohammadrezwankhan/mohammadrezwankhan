# Contributing

Thank you for considering a contribution. This repository represents the public GitHub profile for Mohammad Rezwan Khan, so contributions should improve clarity, accuracy, accessibility, or technical credibility.

## Good Contributions

- Fix typos, broken links, and formatting issues.
- Improve accessibility for tables, links, and images.
- Suggest clearer wording for engineering impact statements.
- Add relevant public evidence, such as project links, research profiles, or portfolio pages.
- Improve maintainability of Markdown, shields, and profile sections.

## Pull Request Checklist

- [ ] The change is accurate and professional.
- [ ] Links are public and working.
- [ ] Markdown renders cleanly on desktop and mobile.
- [ ] No private, confidential, or unverifiable information is added.
- [ ] The profile remains focused on battery systems, BESS, smart grid, power electronics, and engineering reliability.

## Style Guide

- Prefer impact over duties.
- Prefer concise engineering language over buzzwords.
- Keep claims evidence-backed.
- Use tables only when they improve scanning.
- Avoid decorative content that does not support recruiter or collaborator understanding.
